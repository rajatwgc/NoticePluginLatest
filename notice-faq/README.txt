== Installation ==

### **AUTOMATIC INSTALLATION VIA THE PUGLIN LIBRARY**

You can find our plugin at this address: [https://wordpress.org/plugins/notice-faq/](https://wordpress.org/plugins/notice-faq/)

### STEP BY STEP

1. Upload the entire `noticefaq` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the **Plugins** **Menu** (**Plugins > Installed Plugins**)
3. Sign up on notice.studio
4. Create your project (FAQ, blog, doc, and more) and publish it
5. Go to Integrations → WordPress → Copy your ID project
6. Come back to WordPress → add a new block → select Notice
7. Drag and drop the Notice block anywhere you want on your page and paste the ID project
8. Click on Update, and than View page to check your project live

### ELEMENTOR USERS

If you are using [Elementor](https://elementor.com/):

1. On the page you want to implement your content, create a new block by choosing the "shortcut" type of block
2. Copy & paste your shortcode code

For more information, see the detailed manual [here](https://integrations.mynotice.io/wordpress-c2f5) and get ready in less than 5 minutes.

== Screenshots ==

1. Find Notice Dashboard on the left side menu after installing and activating the Notice plugin

2. Connect to your Notice account to start creating your content

3. Choose between ready-to-use templates, type of projects, or start from scratch

4. Intuitive no-code editor - type ‘/’ to find all block possibilities such as paragraphs, bullet lists, images, expandable for FAQs, and much more

5. Create together with our AI assistant that can generate text and images to make your creating flow easier and fast

6. When you’re finished creating, go to Integrations and copy your Project ID - if you use Elementor, copy the second one

7. Add a new block, search for Notice (or Shortcode if you use Elementor), drag and drop the notice block anywhere you want on your page, and paste the ID project inside

8. Click on Update, then View Page, and check that your project is live into your WordPress website

9. Extra features: translate your content in 100+ languages and offer a better experience for your audience

10. Extra feature: gain insights with our Analytics - check visits, location and device info of your users, what they’re looking for on the search bar, and much more
